# Pitch to Producer

这不是任何现有 IP 的同人游戏，也未使用任何现有 IP 的素材、设定或商标。

这是一个原创机制样机，用来验证：

> 玩家不亲自开枪，而是通过自然语言、授权、方针调整和关键节点 override，调度一组半自主战术单位。

## 核心价值

传统战术游戏让玩家逐格移动、逐个点技能。  
本样机验证另一条路线：

- 战术单位会自主作战
- AI 副官能维持基础调度
- 玩家可以零指挥观察
- 玩家也可以用自然语言改写高层战术意图
- LLM 负责理解意图
- 本地规则引擎负责结算战斗
- 单位性格和枪种定位影响执行

## 适配的 IP 类型

它适合以下 IP：

- 玩家身份天然是指挥官
- 角色是可调度、可养成、有强人格的单位
- 玩家和角色存在长期关系
- 战斗适合体现“授权 / 调度 / 约束 / 关键决策”

这不是聊天 NPC，而是 AI-native 指挥玩法。
