# 《少女前线3战斗模块》立项企划书：代理指挥与自然语言战术输入原型

**收件人：** 羽中  
**主题：** 《少女前线3战斗模块》立项企划书：代理指挥与自然语言战术输入原型

---

羽中，

我做了一个关于《少女前线3》战斗模块的原型。

这里的《少女前线3》不是指我知道你们存在这个项目，而是把它作为“少前下一代战斗范式”的假想命名。

核心想法很简单：

**把传统自动战斗升级成“代理指挥官”，同时保留玩家一句话接管关键指挥的能力。**

少前2的战斗有一个病：它看起来想做出 XCOM 式的战术决策层，但实际游玩里，手操成本太高，玩家很容易退回自动战斗。结果就是，决策层存在，但没有真正进入玩家体验。

《圣兽之王》有一个对称的病：它的编队、预设和策略系统很深，但难度压力不够，很多时候照推荐方案也能过。系统深度存在，但没有被充分需要。

两个病，殊途同归：

**战斗系统里明明存在“决策”，但玩家没有以低成本、高反馈的方式真正参与进去。**

我的原型把战斗控制拆成三种模式：

## 一、纯规则自动战斗

碾压局、日常本、低压副本，不需要任何 LLM。

直接状态机跑即可。

这部分就是传统自动战斗，用来保证低成本、低打扰、可快速刷图。

## 二、玩家自然语言指挥

玩家可以直接下一句话：

> 集火残血，前排顶住。

系统把这句话映射成角色状态机指令：

```text
玩家命令 → 命令编译器 → 状态机
```

命令编译器负责把自然语言命令落到角色的目标选择、移动倾向、技能释放、阵线保持等具体行为上。

也就是说，玩家亲自指挥时，不需要再经过一层战术 AI。

玩家本人就是指挥官。

## 三、战术 AI 代理指挥

玩家不想亲自操作时，可以开启战术 AI。

这时战术 AI 替代传统自动战斗，读取战场状态，像代理指挥官一样下命令：

```text
战场状态 → 战术 LLM → 战术命令 → 命令编译器 → 状态机
```

战术 LLM 负责根据战场状态判断应该下什么命令；命令编译器负责把命令落到角色行为上。

传统自动战斗是“角色自己按规则打”。

代理指挥是“系统先判断战局，再给角色下命令”。

它主要解决的是传统自动战斗太弱智的问题，而不是做战场聊天。玩家不是在战场上陪 AI 聊天，而是在需要的时候下战术命令；不想管的时候，则由战术 AI 代理指挥。

我已经做了一个纯文字 demo，并跑了一千局模拟，结果如下：

| 输入 / 模式 | 胜率 |
|---|---:|
| 好指挥：集火残血 | 96% |
| 不指挥 | 33% |
| 莽夫冲锋 | 0.3% |
| 无关闲聊 | 31% |

这个结果说明：

**指挥质量本身会显著改变战局。**

无关闲聊不会被系统洗成好策略。  
错误指挥会带来坏结果。  
正确指挥会带来高胜率。

也就是说，这不是“随便输入一句话就赢”，而是把“指挥”真的做进战斗系统。

我也把对手从木桩换成了会行动的敌方策略，差距仍然存在，甚至更明显。

## 成本估算

这个方向并不像表面看起来那么贵。

原因很简单：它不是每个角色每回合都长篇调用 LLM，也不是战场聊天系统。主要调用发生在“下命令”这个节点。

按当前 demo 的估算：

```text
15 局 / 小时 × 1.5 次指挥 / 局 × 280 tokens ≈ 6,300 tokens / 小时
```

按 deepseek-v4-flash 当前价格，即使粗暴按输出价格 **2 元 / 百万 tokens** 估算：

```text
6,300 tokens / 小时 ≈ 0.0126 元 / 小时
```

也就是一小时约 1 分钱。

即使把上限放宽到：

```text
20,000 tokens / 小时
```

也大约是：

```text
0.04 元 / 小时
```

所以这个模块真正要验证的不是 token 成本，而是：

- 代理指挥是否比传统自动战斗更强
- 玩家一句话指挥是否真的有战术反馈
- 角色状态机能否稳定执行命令
- 战术 AI 的判断是否足够像“指挥官”
- 这套东西能否嵌进少前式战斗系统

## 项目与复现

GitHub 项目已公开：

https://github.com/Corrame/real-time-commander-demo

仓库里有最小可运行 demo、模拟脚本、1000 局结果复现方式，以及网页可视化 demo。

网页 demo 用左右对照展示自然语言指挥和普通 baseline 的行为差异：左侧是自然语言命令控制的战斗单位，右侧是 dumb baseline，用来直观看到“同样战场下，命令如何改变战术行为”。

附件是完整企划说明。  
GitHub 用来证明这个方向不是纯概念，而是已经能跑出可验证差异。

我真正想表达的是：

**《少女前线》一直把玩家称为“指挥官”。**

如果这套机制成立，那么“指挥官”就不再只是剧情身份，而是一个真实的战斗输入接口。

玩家可以亲自下命令。  
玩家不想管时，战术 AI 可以代理指挥。  
低压局也可以完全不开 AI，直接状态机碾过去。

这套东西的重点是：

**把自动战斗升级成代理指挥，把自然语言变成真正的战术输入。**

你想抄走这个想法也可以直接抄。
