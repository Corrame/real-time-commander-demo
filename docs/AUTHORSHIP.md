# Authorship & Proof Boundary

本仓库是 **Real-Time Commander Demo** 的公开署名与证据锚点。

## 证明边界

当前 1.0 版本证明的命题是：

> player natural-language command → command interpreter → finite tactical policy → state-machine execution → measurable battle outcome difference

当前版本不声称是：
- 完整商业游戏
- 完整战术 AI 指挥官
- 最终战斗系统
