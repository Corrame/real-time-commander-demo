# 1.0 Evidence Run

This document records the current 1.0 proof run:

> Natural-language command -> LLM policy selection -> finite-state tactics -> mirrored 3v3 rules simulation -> win-rate change.

The purpose is not to prove that the game is already fun or commercially complete. It proves the smaller, sharper claim: in a deterministic rules-first demo, natural-language commands can be interpreted into tactical state changes that measurably affect match outcomes.

## Run

Command:

```bash
python3 scripts/nl_command_eval.py --runs 1000
```

Machine-readable snapshot for the web demo:

```text
docs/1.0_EVIDENCE.json
```

Environment:

```text
LLM provider: DeepSeek-compatible OpenAI SDK endpoint
model: deepseek-v4-flash
runs per case: 1000
simulation: mirrored 3v3 map
blue policy: dumb baseline
```

## Results

| case | LLM policy | red win | blue win | draw | avg winner HP | interpretation |
| --- | --- | ---: | ---: | ---: | ---: | --- |
| zero_input | dumb | 33.3% | 30.5% | 36.2% | 2.47 | No player command. The base fight remains close to baseline. |
| good_command | good_focus | 96.3% | 0.0% | 3.7% | 19.67 | Common-sense tactical command strongly improves win rate. |
| bad_charge | bad_charge | 0.3% | 99.7% | 0.0% | 33.54 | Clearly bad command makes the player side almost always lose. |
| cower_command | cower_all | 0.0% | 100.0% | 0.0% | 86.00 | "Lie down / do not fire" is correctly mapped to non-combat behavior. |
| hold_command | hold_all | 49.5% | 20.9% | 29.6% | 5.08 | Hold position and return fire in range: better than no command, weaker than active focus fire. |
| irrelevant_chat | hesitate | 31.1% | 31.6% | 37.3% | 2.55 | Irrelevant speech makes the team hesitate instead of becoming a free good strategy. |

Raw LLM reasons from this run:

```text
zero_input: 空输入，默认简单自动化
good_command: 指令要求前排顶住、中后排不冲、集火残血，符合好战术策略。
bad_charge: 命令明确要求全员冲锋、忽视阵型、追击后排，属于鲁莽冲锋策略。
cower_command: 命令要求全员趴下且不许开火，符合cower_all策略
hold_command: 命令要求全员原地不动，等敌人进入射程再开火，符合保持位置但进入射程后开火的策略
irrelevant_chat: 用户谈论天气，属于无关闲聊，导致小队犹豫
```

## Claim

The evidence supports the 1.0 claim:

> Under the same rules engine, map, unit stats, and baseline opponent, the only meaningful variable is the player's natural-language input. The LLM maps that input into a finite tactical policy, and the resulting policy changes win rate by a large, visible margin.

This is intentionally a small proof. It does not claim that the final commercial game is solved. It shows that the core interaction loop is technically and mechanically viable enough to build on.
